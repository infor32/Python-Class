# 2.4 While Loops

### 當這樣時就那樣

```python
i = 0
while i < 10 :
    print(i)
    i += 2
```

###  ​一些小語法

`break`直接跳出當前迴圈

`continue`略過剩下步驟，直接跳到迴圈的下一步

## 小練習

> 請輸出1~100中所有3的倍數

> 請實作一個程式，可以猜出我所想的數字，這個數字是一個介於1~100的正整數，程式每猜一個數字，使用者便可以告訴程式它所猜的數字，大於、小於或等於正確答案，嘗試讓程式以最少的次數猜到答案。
