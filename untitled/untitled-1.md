# 2.2 Operators

「運算子就是把兩個運算元連接在一起的東西」by 03t

### Arithmetic Operators

| Operator | Name | Example |
| :--- | :--- | :--- |
| + | Addition | x + y |
| - | Subtraction | x - y |
| \* | Multiplication | x \* y |
| / | Division | x / y |
| % | Modulus | x % y |
| \*\* | Exponentiation | x \*\* y |
| // | Floor division | x // y |
