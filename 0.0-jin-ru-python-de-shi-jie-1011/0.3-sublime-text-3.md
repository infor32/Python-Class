## 編輯界面

將檔案存成 ".py" 編輯器就會幫你上色啦！還有自動完成喔XD

![Sublime text 3 interface](../.gitbook/assets/image%20%285%29.png)



### 寫完程式後~

1. 在這上面寫好程式後，將檔案存成 .py 的格式
2. 打開你的 cmd ，利用 cd 進入檔案所在的資料夾
3. 打上 " python  檔名.py " 就可以執行囉

大家可以先用剛剛的 Helloworld 程式試試看！
